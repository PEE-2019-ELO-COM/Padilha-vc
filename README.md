O programa funciona da seguinte maneira:

Digita-se o número de 0 a 9 desejado, e a resposta é '1' ou '0' de cada led do display. Ou seja,
caso eu digite o número '1' a resposta do programa será '0 1 1 0 0 0 0', visto que apenas os leds 'b' e 'c'
serão acesos.  

O programa foi realizado utilizando mapas de karnaugh.
